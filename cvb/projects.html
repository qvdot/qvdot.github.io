Detection of tumors and classification of those into malignant and non-malignant using Deep Learning in cranial tomography scans (Huancayo, Peru)

This project plans to use a Convolutional Neural Network (CNN) and other networks for recognizing brain tumors using tomography scans and based on previous research in the area. The project aims to apply Transfer Learning on the previous network to build a network that can determine if the brain tumor is malignant.
